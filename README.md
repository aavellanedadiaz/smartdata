## 📖 A cerca del proyecto
Este proyecto contiene el codigo fuente para la Extraccion, Transformacion y Carga de datos utilizados para identificar posibles problemas mentales y salud mental en pacientes.

### Construido con
* Azure Databricks

## 🚀 Componentes
El repositorio esta dividido en diferentes carpetas que contienen los componentes utilizados para el proceso de Ingestion, Transformacion y Carga. Ademas del flujo y seguridad de los Datos.

1. Proceso: Contiene los componentes (Notebook) involucrados en la extraccion, transformacion y carga de los Datos.
2. Evidencias: Contiene informacion sobre el procesamiento de los Datos y los servicios cloud aprovisionados.
3. Seguridad: Contiene informacion sobre las reglas de seguridad aplicadas sobre los componentes del catalogo.
4. Datasets: Contiene los archivos fuente (json - csv)
5. PrepAmb: Contiene un notebook que garantiza la creacion de objetos de bases de datos y componentes de Unity Catalog.
6. Reversion: Contiene un notebook para eliminar los componentes de bases de datos creados durante el proceso.
7. Certificaciones: Contiene informacion de las certificaciones llevadas a cabo durante el curso.
